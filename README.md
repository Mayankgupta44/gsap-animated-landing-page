# GSAP Animated Landing Page

A modern animated landing page built using GSAP (GreenSock Animation Platform) to create smooth, interactive UI animations.

## Features
- Smooth scroll-based animations
- Image transitions and hover effects
- Dynamic background gradient changes
- Custom cursor interactions
- Responsive layout

## Tech Stack
- HTML
- CSS
- JavaScript
- GSAP

## Purpose
This project was built to explore advanced frontend animations and improve UI interaction skills using GSAP.
Built as part of frontend practice and assignment-based work.

## Learnings
- Worked with GSAP timeline and scroll triggers
- Improved understanding of animation timing and easing
- Built interactive UI elements using JavaScript
